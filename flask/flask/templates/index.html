Crop Yield

Crop Yield refers to the production of Crops in a given area.In this project an attempt is made to predict the yield of crop with high accuracy and minimum error. The Crop yield Prediction is done with the help of Decesion tree.Three algorithms Random Forest,Decesion tree,GradientBoostingRegressor are used to calculate the yield.

Accuracy Table
ML Algorithm Accuracy
Random Forest 75.26
Decesion tree 92.04
GradientBoostingRegressor 67.99
